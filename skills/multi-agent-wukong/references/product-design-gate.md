# Product Design Gate

Public rule set:

- applies to every `S/M/L/XL` design-start task for `Role=Chang'e`
- requires real Product Design invocation evidence
- requires plugin id/version, tool-call id, timestamp, input digest, artifact hashes, confirmation evidence, and dependency status
- no automatic Figma or Claude Design fallback
- exact fail-closed status for unavailable runtime: `BLOCKED_PRODUCT_DESIGN_PLUGIN_UNAVAILABLE`
